Documentation

What is it?

WatoCMS is a Nodejs Content Management System. It stores your articles and resources into a Redis database for quick access, then compiles HTML to send to a visitor

How use it?

If you have successfully installed node and WatoCMS on this server, then you're doing pretty good.

The next steps are to:

  • Go to www.yourdomain.com/auth/index.htm to setup user accounts
    • The default username is "wato" and the password is "password"
    • Delete this default account after you have created your own
  • Then, start creating articles
  • Sprinkle on some CSS with the CSS editor tool
  • Finally, edit your settings so to have a default article
